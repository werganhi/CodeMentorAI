# CodeMentor AI

Мобильное Flutter-приложение для дипломной работы на тему:

**"Разработка мобильного приложения для обучения программированию с элементами искусственного интеллекта"**.

## Запуск

```powershell
flutter pub get
flutter run
```

## Основные функции

- уроки по основам Python и алгоритмам;
- теория, примеры кода и тесты;
- практический тренажер;
- справочник терминов;
- локальный ИИ-наставник;
- XP, прогресс и стрик обучения;
- светлая и темная тема.

Документы находятся в папке `docs/`.
