# CodeMentor AI

Тема проекта: **разработка мобильного приложения для обучения программированию с элементами искусственного интеллекта**.

## Что внутри

- Flutter-приложение `CodeMentor AI`.
- Уроки по Python, алгоритмам, функциям, коллекциям, отладке и ИИ.
- Практический тренажер с проверкой ответов.
- Справочник терминов программирования.
- Экран ИИ-наставника с локальной имитацией AI-диалога для дипломного прототипа.
- Сохранение прогресса, XP, стрика и имени студента через `shared_preferences`.
- Отдельные документы:
  - `docs/Дипломная_документация.md`
  - `docs/Техническое_задание.md`

## Запуск

```powershell
flutter pub get
flutter run
```

Если Flutter попросит создать платформенные файлы в папке проекта:

```powershell
flutter create .
flutter pub get
flutter run
```

## Примечание по ИИ

В текущем готовом прототипе ИИ-наставник работает локально: он анализирует ключевые слова запроса и возвращает заранее продуманные учебные подсказки. Это сделано, чтобы приложение запускалось сразу без API-ключей и интернета. В дипломной документации описан вариант расширения через внешний API языковой модели.
